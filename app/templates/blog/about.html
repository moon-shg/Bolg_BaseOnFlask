Blog Introduce

{# content #}

  • 后端使用 Flask 框架，采用 Jinja2 模板引擎渲染前端模板。并实现 RESTful API， 提供文章发表、修改、评论等多个接口。

  • 前端 Web 模板基于 Bootstrap 4 搭建， 实现响应式布局。同时使用 HTML5/CSS3/jQuery 等技术。

  • 数据库采用 MySQL8.0，使用 SQLAlchemy ORM框架(通过 Flask-SQLAlchemy 插件集成)。

  • 项目基于 CentOS7 + uWSGI + Nginx 部署于云服务器上。

  • 发布文章支持 CKEdiotr 和 Markdown 两种富文本编辑器。发布内容可在客户端或服务器端实现 Markdown 到 HTML 转换。（通过 Bootstrap-Markdown, Flask_CKEditor 插件以及 markdown, bleach 模块集成）

  • 使用 Whoosh 和 Jieba 实现中文全文检索(通过 Flask-WhooshAlchemyPlus 插件和 Jieba 中文分词库集成)

  • 支持本地邮箱注册账户。域名备案后计划通过 OAuth2 实现第三方登录。

  • 用户登录后，支持个人资料编辑，头像上传和更换。同时支持发表文章评论，回复评论，编辑评论以及收藏文章。管理员权限用户可封禁不良评论。

  • 发表的文章支持多级分类。并且支持日期归档，热门文章，随机文章。

  • 支持社交平台分享，如分享到微信、QQ、微博等平台。
